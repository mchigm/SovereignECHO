# Song-Speech Conversion

A model that converts between melodic vocal features and speaker voice features

--- 

This repository is from our paper, Feature Engineering for Speaker Characterization in Melodic Vocalization [DOI]().  Published to IEEE ACAI 2025.  It aims to convert the melodic vocal feature to speaker vocal feature.
